# 🚀 AzurePrep AZ-900

Una aplicación web interactiva diseñada para ayudarte a estudiar y superar el examen de certificación **Microsoft Azure Fundamentals (AZ-900)**. 

Esta herramienta convierte el estudio en una experiencia dinámica a través de la gamificación, permitiéndote poner a prueba tus conocimientos sobre la nube de Microsoft de una manera entretenida.

## ✨ Características Principales

* **Tarjetas de Estudio (Flashcards):** Repasa conceptos clave de Azure rápidamente.
* **Retos y Cuestionarios:** Pon a prueba lo que has aprendido con preguntas similares a las del examen oficial.
* **Sistema de Gamificación:** Gana puntos de experiencia (XP) y desbloquea logros a medida que avanzas en tu estudio.
* **Interfaz Moderna:** Diseño atractivo en modo oscuro para reducir la fatiga visual durante largas sesiones de estudio.
* **100% Front-end:** Construido con HTML, CSS y JavaScript puro, sin necesidad de servidores complejos.

## 🛠️ Tecnologías Utilizadas

* HTML5
* CSS3 (Variables, Flexbox/Grid, Animaciones)
* JavaScript (Vanilla JS)
